---
title: Fee Structure
---

FTA has a moderately complex but easily understandable fee structure that offers significant flexibility for builders to set fees that accurately represent their costs of doing business.

In order to obtain a FTA jump permit (one-time use pass for a gate), the user must first pay a fee in EVE tokens. The fee consists of several parts:
1. A [Gate Fee](./Gate%20Fee.md) for the source gate
2. A [Gate Fee](./Gate%20Fee.md) for the destination gate
3. A [Network Node fee](./Network%20Node%20Fee.md) for the source gate
4. A [Network Node fee](./Network%20Node%20Fee.md) for the destination gate
5. A [bounty pool fee](./Bounty%20Pool%20Fee.md)
6. A [developer pool fee](./Developer%20Fee.md)

That fee is then scaled according to the requested [duration of the jump permit](.//Permit%20Duration.md) (how long it's valid for) and any [blacklist penalties](../Penalties/Blacklist.md) associated with the character the permit is for.
