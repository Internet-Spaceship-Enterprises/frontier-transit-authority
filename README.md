# Frontier Transit Authority

The Frontier Transit Authority (FTA) is a project to build a community-managed gate network throughout the EVE Frontier universe.

You can access the dApp [here](https://internet.spaceship.enterprises/fta/).

You can access the full documentation [here](https://internet.spaceship.enterprises/fta/docs/).

A demo video is available [here](https://www.youtube.com/watch?v=cZjk4wTQV3M).
