Once the FTA has expanded enough to allow for multi-system routes, it will be important to offer a route planning tool that allows travelers to see the shortest and/or lowest-cost route from A to B using the FTA network. We intend to implement this in our dApp UI.

The UI will not only support route planning, but will also support the quoting and purchasing of "route permits", which are a series of Jump Permits for all of the gates in your planned route. Being able to purchase a Route Permit will eliminate the need to open the dApp interaction for each gate along the route, instead allowing the traveler to simply jump through each gate.
