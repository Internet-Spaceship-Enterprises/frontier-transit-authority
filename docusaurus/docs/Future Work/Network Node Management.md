Currently, since depositing fuel into network nodes can only be done in-game (requires a sponsored transaction), builders must retain ownership of their network nodes to be able to keep them fueled and operational.

Ideally, FTA would obtain ownership of the network nodes that power the FTA-managed gates in order to maximize the automation that can be done (e.g. offlining a network node when no one is using the gates, to save fuel). Once CCP implements the ability for players other than the owner of a network node to deposit fuel into it (and receive compensation for doing so), we can modify the design to enable FTA-owned network nodes.
