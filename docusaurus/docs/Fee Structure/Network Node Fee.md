Network Node fees are fees associated with using the network node that powers an FTA-managed gate. They are configured and managed by the builder of the network node, or their delegated representative.

Network node fees are specific to a single gate, not a pair of linked gates. When a jump permit fee is calculated, the network node fees for both the source and destination gates are included. This per-gate fee structure allows for future development of a dynamically-linked, reconfigurable gate network.
